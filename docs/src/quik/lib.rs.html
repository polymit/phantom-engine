1//! # Quik: High-Fidelity Transport Layer
2//!
3//! `quik` is a specialized HTTP transport engine designed for absolute network identity parity
4//! with Google Chrome. Unlike general-purpose HTTP clients, `quik` provides low-level control
5//! over the entire protocol stack—from TLS handshakes to HTTP/2 frame signaling—to ensure
6//! that every network interaction is indistinguishable from a real browser.
7//!
8//! ## Core Capabilities
9//! - **TLS Identity**: Uses BoringSSL to replicate Chrome's ClientHello, including GREASE,
10//!   extension permutation, and post-quantum key shares.

12//!   window deltas to match Chromium's behavioral fingerprints.
13//! - **Stealth Navigation**: Manages request context metadata (sec-fetch headers) and HPACK
14//!   sensitivity to bypass advanced anti-automation heuristics.
